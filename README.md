# 玖富国际公众号消息设置

  对用户开放公众号消息设置开关，方便用户选择性接收自己关注的消息类型。公众号入口菜单届时由运营进行配置

### Usage

启动本地服务

``` bash
$ npm start 
```

打包编译

``` bash
$ npm run build
```

打包编译 - 调试包

``` bash
$ npm run build:debug
```

### 说明
>`VUE_APP_COMMON_CONFIG` 公共配置文件地址，开发、测试环境可自由修改，生产环境配置为服务器根目录地址
***
>`VUE_APP_CONSOLE` 是否显示vscode标识,**生产环境**应为`false`
